# 臺羅拼音轉換工具

此網站由 GitHub Pages 提供，瀏覽器載入 `index.html` 和
`web_database.js` 後即可運作；不需要後端服務。

## 更新對照資料

1. 將最新版檔案放入 `source/`，並維持以下檔名：
   - `漢字臺羅對照表20260723.xlsx`
   - `IPA對照表20260723.xlsx`
2. 執行 `python3 build_web_database.py`。
3. 確認輸出的條目數與抽樣轉換結果後，提交 `web_database.js`、
   `source/` 和建置程式。

漢字檔必須有「漢字」欄；第二欄為臺羅資料（可以命名為「臺羅」，
也可以如 2026-07-23 版保留空白欄名）。讀音行可使用 `文：`、`無：`、
`白：`、`替：`、`俗：` 分類。

IPA 檔使用「2026修訂」工作表：第 7 列的 `TL` 起始欄和前六列的
俄、義、西、德、法、英資料必須保留。網頁採每種語言的最長字串匹配。
其中標示為「拼字」的 `ə、ɚ、ɜ、ɝ` 會原樣保留，不轉成其他符號。
